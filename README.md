# Kugrow OS

AI-powered business operating system for African SMEs.

## Vision

Kugrow OS helps businesses manage:

- POS and sales
- Inventory and stock
- Expenses and finance
- Supplier balances
- Customer credit
- Forecasting and alerts
- Compliance
- AI chatbot and business advice

The first version focuses on retail and wholesale businesses in Zambia.

## Tech Stack

Frontend:
- React
- Vite
- Tailwind CSS
- Axios
- React Router

Backend:
- Django
- Django REST Framework
- PostgreSQL
